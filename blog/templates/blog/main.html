Telegram BOT

Телеграм бот

Ідея була зробити бота помічника на всі руки для клану в ігрі Clash Royale. До створення бота, голова клану користувався Google sheets до ручного запису всього що відбувалось в клані. Бот використовувався на практиці, але не був завантажений на жодний хостінг. На даний момент бот є відключеним та не використовується. Можливості бота:

  • Збір інформації про ігроків клану.
  • Аналіз інформації та її обробка.
  • Формування інформації в доступному вигляді.
  • Контакт з ігроками, надавання інформації через бота.
  • Аналіз ігроків та надання порад (підвищити або вигнати).

Використані бібліотеки та інструменти в проекті:

  • Python - як основна та єдина мова.
  • BeautifulSoup - web scraping.
  • SQLite, як основна база данних.
  • Plotly - більш сучасний аналог matplotlib.
  • Telegram Bot API - бібліотека на якій базувався бот.

Python

Час навчання: 12+ місяців.

Після самостійної проби вникнути в програмування через вивчення Java, мова програмування Python для мене було порятунком. На універі більше був уклон на різні алгоритми та продуктивність, натомість різний функціонал, включно з ООП вивчався окремо. Список практикуємих інструментів:

JavaScript

Час навчання: 2+ місяців.

При роботі з CSS та HTML виникла необхідність в сторонньому інструменті для створення сайта, конкренто потрібний був більш гнучкий інструмент для анімації. Звісно JavaScript був ідеальним кандидатом. Мої знання по JS на даний момент являються базовими, я пройшов декілька інтернет курсів, дойшовши до ООП.
