Telegram BOT

Ідея була зробити бота помічника на всі руки для клану в ігрі Clash Royale. До створення бота, голова клану користувався Google sheets до ручного запису всього що відбувалось в клані. Бот використовувався на практиці, але не був завантажений на жодний хостінг. На даний момент бот є відключеним та не використовується. Для подальшого використовування буде вимагатись актуалізація.

Можливості бота:

  • Збір інформації про ігроків клану.
  • Аналіз інформації та її обробка.
  • Формування інформації в доступному вигляді.
  • Контакт з ігроками, надавання інформації через бота.
  • Аналіз ігроків та надання порад (підвищити або вигнати).

Використані бібліотеки та інструменти в проекті:

  • Python - як основна та єдина мова.
  • BeautifulSoup - web scraping.
  • SQLite, як основна база данних.
  • Plotly - більш сучасний аналог matplotlib.
  • Telegram Bot API - бібліотека на якій базувався бот.

Travian Troop Calculator

Travian Calculator це міні алгоритм з графічною оболочкою для браузерної ігри Travian. Проект був створюваний на прохання друга. Задачою було обрахувати скільки потрібно ресурсів для створення певної кількості воїнів певного типу, певної фракції.

Можливості програми:

  • Можливість міняти тип та фракцію воїна.
  • Обчислення додаткової інформації.
  • Взяття під увагу додаткові бонуси.
  • Різні мови програми.
  • Можливість створення, як .exe так і .apk версії.

Використані бібліотеки та інструменти в проекті:

  • Python - як основна та єдина мова.
  • Kivy and KivyMD - времворк для написання графічної складової проекта.
  • SQLite, як основна база данних.
