Telegram BOT

The idea was to make a helper bot for the clan in the game Clash Royale. The bot was used in practice, but was not deployed on hosting. The bot is currently disabled and not in use. An update will be required for further use.

Bot features:

  • Collecting information from internet.
  • Analysis of information and processing.
  • Contact with players and provide information.

Libraries and tools used in the project:

  • Python - main programing language.
  • BeautifulSoup - web scraping.
  • SQLite - main database.
  • Plotly - more modern analogue of matplotlib.
  • Telegram Bot API - the library on which the bot was based.

Travian Troop Calculator

Travian Calculator is a mini algorithm with a graphical shell for the Travian browser game. The project was created at the request of a friend. The task was to calculate how many resources are needed to create a certain amount soldiers of a certain type, a certain faction.

Program features:

  • Ability to change the type and faction of the warrior.
  • Calculation of additional information.
  • Taking into account additional bonuses.
  • Different languages.
  • Ability to create both .exe and .apk versions.

Libraries and tools used in the project:

  • Python - main programing language.
  • Kivy and KivyMD - graphic component of project.
  • SQLite - main database.
